# DesignRequest

Минимальная учебная версия приложения для учёта заявок на дизайн-проекты.

## Что оставлено в проекте

- `main.py` — графический интерфейс (Tkinter)
- `database.py` — работа с SQLite (CRUD)
- `config.py` — настройки и русские тексты интерфейса
- `design_requests.db` — база данных
- `.gitignore`
- `README.md`

## Требования

- Python 3.8+
- Tkinter (обычно уже входит в Python на Windows)

## Запуск

```bash
python main.py
```

## Как добавить заявку

1. Заполните поля `Имя клиента` и `Тип проекта` (обязательные).
2. Нажмите кнопку `Добавить заявку`.

## Примечание

Если файл `design_requests.db` отсутствует, он будет создан автоматически при запуске.
